# 设计规范

## 视觉方向

- 主色：浅粉系（背景与弱强调；避免紫粉霓虹、深色仪表盘风）
- 气质：简洁、直观；首屏一眼能扫完历史

## 首屏结构（唯一主场景）

首屏只做「历史列表」这一件事，包含：

1. 轻量标题 / 品牌名（可很小）
2. 搜索框（单一输入）
3. 按规则排序的历史卡片流

禁止：统计条、多栏仪表盘、设置重门户、卡片外再套大卡片墙。

## 卡片层级

每张卡片展示：

- 类型暗示（文字摘录 / 图片缩略图）
- 时间（相对或短日期时间）
- 操作：置顶、删除；点击主体 = 写回剪贴板

置顶区在列表上方；非置顶区时间降序。组内仍降序。

## 交互

- 点击卡片主体 → 写回剪贴板（不自动打到其他窗口）
- 置顶 / 删除为显式按钮，避免误触
- 空状态：一句短文案提示「复制任意内容后将出现在这里」

## 动效（克制）

- 新条目进入：轻微出现即可（1 种）
- 删除：淡出或直接移除（1 种）
- 不做粒子、辉光、大面积渐变动画

# 需求摘要

> 权威源：[PRD.md](../PRD.md)。本文只做会话内快查，冲突时以 PRD 为准。

## 产品定位

面向不懂代码的 Windows 用户，提供本机历史剪贴板，一键找回最近复制的文字/图片并再次粘贴。

## MVP 功能（优先级）

| # | 功能 | 优先级 |
|---|------|--------|
| 1 | 自动监听剪贴板（文字 + 图片）并写入本机历史 | P0 |
| 2 | 历史卡片时间降序；点击写回剪贴板 | P0 |
| 3 | 删除 / 置顶 | P0 |
| 4 | 关键词搜索（仅文字） | P0 |
| 5 | 浅粉底简洁 UI | P1 |

## 数据与生命周期

- 只存本机；不上云
- **关闭窗口 → 系统托盘后台继续**，历史保留、继续监听
- 托盘「退出」或关机后清空
- 超过 24 小时的条目自动丢弃（双保险）
- 条目上限约 100

## 验收对照（摘要）

| 能力 | 标准 |
|------|------|
| 文字 | 复制后 ≤1s 出现在列表顶，内容一致 |
| 图片 | 有缩略图卡片；点击后可再粘贴为图 |
| 排序 | 置顶区在上；组内时间降序 |
| 再复制 | 写回系统剪贴板，用户 Ctrl+V |
| 删除 / 清空 | 删除立即生效；关窗后记录仍在；托盘退出或关机后再开为空 |
| 搜索 | 只匹配文字；图片不参与 |

## 明确不做

云同步 / Supabase / Vercel 部署 / 富文本与文件路径 / 跨开机归档 / 磁盘持久化 / 多平台 / 粘贴到前台窗口。
